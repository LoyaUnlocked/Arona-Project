# Arona Project 
An AI Assistant specialized for **Arch Linux**.

## Features
- **Personality:** cheerful, playful.
- **LLM:** Powered by `Ollama` and `lama3.2:1b`.
- **Environment:** "Optimized" for Hyprland on Arch Linux.

## How to Run
1. Install Ollama: `sudo pacman -S ollama`
2. Run the script: `python arona.py`
